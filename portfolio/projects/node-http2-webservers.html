Node HTTP/2 webservers

node-http2-webservers is a repository on Github containing a readme with a compatability table for HTTP/2 support in various NodeJS frameworks. The repository also contains examples of the frameworks that do have support for HTTP/2, be it native or via the SPDY package.

I created this project to get a clear view of how many frameworks are able to implement the HTTP/2 protocol. I think it is important to see how many frameworks are implementing new/updated protocols, HTTP/2 may become the standard for webservers and frameworks need to get with the time in order to stay relevant. You woudn’t want to work with a framework that is outdated right?

I’m planning to use this project to have a compatability table of HTTP/3 supported frameworks when that’ll come out.
