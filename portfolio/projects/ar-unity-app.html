AR Unity app

C#UnityAndroidiOS

Our project group was tasked with making a concept application, using an emerging technologie, to make it more accessible for citizens to share environment ideas & proposals with the municipality.

We have experimented with different technologies, and chose Augmented Reality because it would best fit our application. We decided to use Unity in combination with Vuforia so that we could make our application easily work with both Android and iOS.

For this project we were asked to make a Proof of Concept, so we made a concept app that we can demo to show the potential that it could have.

This project uses the following toolchain:

  • Unity3D
    • Android Build Support module
    • iOS Build Support module
  • Vuforia
