HR openday app

Android

The openday android app was made as a project for Hogeschool Rotterdam. The purpose of this project is to learn how to create a sufficient android app that meets the requirements of the product owner and the target audience.

The app has a list of open days with the date, when click on an open day you get see the description of the open day, the events that take place, an “add to calendar” button and a share button. There is also a page to get more information about the school institute with a button to the website, and a page where you can ask a question if you have one.

I learned how to make a persona and create a design based on said persona. I learned how to create a test plan and how to user test every feature.
