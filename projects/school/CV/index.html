Mijn Interesses

Ik ben op gegroeit als een Nintendo fan, mijn eerste product van Nintendo is de GameBoy Advanced SP die ik kreeg toen ik zes werd. Toen ik naar de middlebare school ging wilde ik een laptop voor mijn verjaardag, niet veel tijd daarna ben ik in mijn vrije tijd computer games gaan spelen.

Toen ik 14 jaar was speelde ik vooral Minecraft, een game gemaakt in Java waar je alles in kan bouwen. Eerst vond ik het leuk om zonder mods te spelen, maar ontdekte dat het spel veel leuker is met mods. Ik was zelfs geïntereseerd om er één te maken. Daar is mijn interesse voor programmeren begonnen.

Wat ik belangrijk vind

Met programmeren vind ik het heel belangrijk dat er een duidelijk doel is, maar de code die ik schrijf moet makkelijk aantepassen zijn. Verder vind ik het belangrijk dat ik het leuk vind om aan een project te werken, als ik met tegenzin aan de slag ga wordt het niet iets moois.
